Penelitian Sistem Informasi Seminar Tugas Akhir Berbasis Web

Penelitian ini menemukan bahwa sistem informasi seminar tugas akhir berbasis web dapat mengatasi beberapa kendala yang sering terjadi dalam pengelolaan tugas akhir, antara lain:

  • Kesulitan dalam mengatur jadwal seminar
  • Penyampaian informasi tugas akhir yang tidak merata
  • Kesulitan dalam mencari data tugas akhir

Sistem informasi seminar tugas akhir berbasis web ini dibangun dengan menggunakan HTML, CSS, JavaScript, bootstrap dan adminlte. Sistem ini memiliki beberapa fitur utama, antara lain:

  • Manajemen jadwal seminar
  • Penyampaian informasi tugas akhir
  • Pencarian data tugas akhir

Hasil penelitian ini menunjukkan bahwa sistem informasi seminar tugas akhir berbasis web dapat meningkatkan efisiensi dan efektivitas pengelolaan tugas akhir mahasiswa.
